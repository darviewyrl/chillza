# CHILL ZA Website Redesign

The current website feels like an AI-generated landing page where PNG assets were simply placed on the screen.

I want a complete redesign of the visual direction and hero section.

Do NOT make small adjustments to the existing layout.

Instead, redesign the website as if it were a real commercial beverage brand website.

## Design References

Use inspiration from:

- Coca-Cola product pages
- Fanta campaigns
- Schweppes landing pages
- Lipton product marketing
- Modern product landing pages from Dribbble
- Premium beverage advertising websites

Avoid typical AI-generated landing page layouts.

---

# Main Design Goals

The website should feel:

- Professional
- Fresh
- Modern
- Youthful
- Commercial
- Product-focused

When visitors open the website, they should immediately understand:

1. What the product is
2. What flavor it is
3. How much it costs

# Asset Usage Rules

Current assets feel randomly placed.

Fix this.

## Product Can

- Increase size significantly
- Use as hero visual
- Add realistic shadow
- Add depth
- Add subtle floating effect

## Mascot Character

- Reduce size by approximately 40%
- Position beside the can
- Treat as supporting brand mascot
- Do not allow mascot to visually overpower the product

## Mango Elements

- Use as composition elements
- Place partially behind the can
- Create depth
- Avoid random floating fruit

## Ice Elements

- Use only where necessary
- Integrate naturally
- Avoid decorative clutter
