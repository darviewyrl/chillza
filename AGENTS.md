# AGENTS.md

## Project

CHILL ZA – Mango Soda Sparkling Tea Promotional Website

A campaign-style product landing page created for a school advertising project.

The website should feel like a real beverage marketing campaign rather than a startup landing page.

Target audience:

* Teenagers
* Students
* Young adults
* Casual beverage consumers

Brand personality:

* Energetic
* Refreshing
* Friendly
* Playful
* Commercial-quality

---

## Core Design Philosophy

This is NOT a SaaS website.

This is NOT a dashboard.

This is NOT a generic AI-generated landing page.

The experience should feel like:

* Fanta campaign website
* Coca-Cola promotional microsite
* Lipton product campaign
* Beverage commercial landing page

Focus on:

1. Product
2. Visual impact
3. Brand recognition
4. Marketing presentation

---

## Development Rules

Before making visual changes:

1. Compare against approved design directions.
2. Preserve visual hierarchy.
3. Verify desktop and mobile separately.
4. Test responsive behavior.

Never:

* Add emojis
* Use placeholder graphics
* Use stock illustrations
* Create dashboard-style cards
* Add AI-style floating decorations

Always:

* Use Lucide icons
* Use consistent spacing
* Keep typography clean
* Maintain premium commercial feel

---

## Accessibility

Requirements:

* Semantic HTML
* Accessible buttons
* Keyboard navigation
* Responsive design
* Proper image alt text

---

## Quality Standard

Every section should answer:

"Would this look believable as a real beverage campaign website?"

If the answer is no, redesign it.
