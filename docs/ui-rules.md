# UI Rules

## Visual Style

Theme:

* Mango Yellow
* Warm White
* Deep Black
* Fresh Green accents

Mood:

* Refreshing
* Bright
* Energetic
* Commercial

---

## Hero Rules

The hero is the most important section.

Visual hierarchy:

1. Product can
2. Hero title
3. Price badge
4. Mascot
5. Supporting text

Never reverse this order.

---

## Typography

Headlines:

* Large
* Bold
* High contrast

Body text:

* Easy to read
* Comfortable spacing
* Moderate line length

Avoid:

* Overly condensed text
* Excessive letter spacing
* AI-style oversized random headings

---

## Components

### Buttons

Style:

* Rounded
* Premium
* High contrast

Never use:

* Neon gradients
* Emoji icons
* Generic AI button styles

---

### Cards

Use cards only when necessary.

Avoid:

* Dashboard layouts
* Repeated identical blocks
* Empty containers

Prefer:

* Editorial layouts
* Storytelling layouts
* Product-focused layouts

---

## Icons

Allowed:

* Lucide Icons
* Official brand icons

Not allowed:

* Emoji
* Clipart
* Decorative icon packs

---

## Mobile Rules

The mobile layout is not a scaled-down desktop layout.

Use:

* Approved mobile artwork
* Mobile-specific spacing
* Mobile-specific composition

Always verify:

* No overflow
* Readable typography
* Comfortable touch targets

---

## Final Quality Check

Before merging:

* Compare against approved mockups
* Check desktop
* Check mobile
* Check visual hierarchy
* Check spacing
* Check typography

If any section feels AI-generated, redesign it.
